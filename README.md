[winston](https://github.com/winstonjs/winston) Playground
(with [winston-daily-rotate-file](https://github.com/winstonjs/winston-daily-rotate-file)).

## Command

### Build

```bash
$ yarn run build
```

### Debug

```bash
$ yarn run dev:run
```

Use `ts-node-dev`:

```bash
$ yarn run dev:debug
```

### Format

```bash
$ yarn run format:fix
```
